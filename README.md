# Cloud-computing-and-Azure-Data-Fundamentals-repository

# ☁️ Microsoft Azure Project – Cloud Computing & Data Fundamentals

## 📘 Overview
This project was completed during a **Data Technician Bootcamp**, focusing on **cloud computing fundamentals, data governance, and Azure services**.

The project demonstrates an understanding of how organisations can leverage **Microsoft Azure** to securely store, manage, and analyse data while ensuring compliance with **data protection regulations**.

---


### ☁️ Cloud Computing Fundamentals
- Understanding of **cloud computing concepts** and real-world applications  
- Knowledge of cloud service models:
  - **IaaS (Infrastructure as a Service)** – Virtual machines and infrastructure control  
  - **PaaS (Platform as a Service)** – Managed development environments  
  - **SaaS (Software as a Service)** – Cloud-based applications (e.g., Microsoft Teams)  
- Awareness of deployment models:
  - Public Cloud  
  - Private Cloud  
  - Hybrid Cloud  

✅ Benefits of cloud computing:
- Cost efficiency (**pay-as-you-go model**)  
- Scalability and flexibility  
- Remote access and collaboration  
- Reduced infrastructure management 

---

## 🔐 Data Governance & Regulations

### 📌 Key Regulations Covered
- **GDPR (General Data Protection Regulation)**  
- **Data Protection Act 2018 (UK)**  
- **Computer Misuse Act 1990**  
- **Copyright and Data Privacy Laws**

---

### 🔍 Data Governance Principles
- Secure handling of personal and sensitive data  
- Data minimisation and purpose limitation  
- Ensuring data accuracy and retention policies  
- Access control and data security  

✅ Examples of sensitive data:
- Personal details (name, address, DOB)  
- Health information  
- Financial data  

✅ Risks addressed:
- Unauthorised access  
- Data breaches  
- Misuse of information   

---

## ⚙️ Azure Service Recommendations

### 📊 Data Storage Solutions
- **Azure SQL Database**
  - Structured data storage (sales, customers)
  - Strong relational database capabilities

- **Azure Blob Storage**
  - Scalable storage for large datasets (images, files)
  - Cost-effective for unstructured data

---

### 🔄 Data Integration & Processing
- **Azure Data Factory**
  - Automates data pipelines (ETL processes)
  - Integrates data from multiple sources

---

### 📈 Data Analytics & Insights
- **Azure Synapse Analytics**
  - Large-scale data analytics and reporting
- **Azure Machine Learning**
  - Predictive analytics and customer behaviour insights

---

### 🔐 Security & Reliability
- Built-in **encryption and access control**
- Backup solutions:
  - **Azure Backup**
  - **Azure Site Recovery**

✅ Ensures business continuity and data protection
